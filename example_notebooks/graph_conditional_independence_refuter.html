Testing Assumptions in model with DoWhy: A simple example

This is a quick introduction to how we can test if our assumed graph is correct and the assumptions match with the dataset. We do so by checking the conditional independences in the graph and see if they hold true for the data as well. Currently we are using partial correlation to do that.

First, let us load all required packages.

%autoreload 2

import numpy as np

import os, sys

Step 2: Input causal graph

We now input a causal graph. You can do that in the GML graph format (recommended), DOT format or the output from daggity - To create the causal graph for your dataset, you can use a tool like DAGitty that provides a GUI to construct the graph. You can export the graph string that it generates.

[4]:

graph_string = """dag {
        W0 [pos="-2.200,-1.520"]
        W1 [pos="-1.457,-1.533"]
        W2 [pos="-0.763,-1.547"]
        W3 [pos="1.041,-1.587"]
        W4 [pos="1.510,-1.560"]

model=CausalModel(

Step 4: Testing for Conditional Independence

We can check if the assumptions of the graph hold true for the data using model.refute_graph(k, method_name,independence_constraints) We are testing X ⫫ Y | Z where X and Y are singular sets and Z can have k number of variables. Currently we are using “partial_correlation” method by default and k value is 1 by default unless input.

[8]:

refuter_object = model.refute_graph(k=2, method_name = "partial_correlation")  #Change k parameter to test conditional independence given different number of variables

[9]:

print(refuter_object)

Method name:partial_correlation
Number of conditional independencies entailed by model:483
Number of independences satisfied by data:396

Testing for a set of edges

We can also test a set of conditional independences whether they are true or not The input has to be in the form - [( x1, y1, (z1, z2)), ( x2, y2, (z3, z4)), ( x3, y3, (z5,)), ( x4, y4, ()) ]

[10]:

print(model.refute_graph(method_name = "partial_correlation",independence_constraints = [('W3', 'Z0', ()), ('W0', 'Z0', ()),('W3', 'Z0', ('W1','W2',))]))

Method name:partial_correlation
Number of conditional independencies entailed by model:3
Number of independences satisfied by data:3
Test passed:True
